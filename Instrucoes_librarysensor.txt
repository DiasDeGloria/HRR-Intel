Ativar I2C no Raspberry Pi
Habilite a conexão I2C do Raspberry Pi usando raspi-config.
$ sudo raspi-config
"Opções de Interface" ⇒ "P5 I2C" ⇒ "Sim"

Confirmação de comunicação com VL53L0X e confirmação do canal I2C

$sudo apt-get update
$sudo apt-get upgrade
$ sudo apt-get dist-upgrade
$sudo apt-get install python-dev and python3-dev

$sudo apt-get -y install python-smbus i2c-tools
$sudo i2cdetect -y 1

Baixar, descompactar e copiar a API VL53L0X API p/ rasp
<https://www.st.com/content/st_com/en/products/embedded-software/proximity-sensors-software/stsw-img005.html>

Compile a API VL53L0X no Raspberry Pi

O makefile para compilar a API VL53L0X no Raspberry Pi está disponível em
 "https://github.com/cassou/VL53L0X_rasp",
 portanto, baixe-o em seu diretório inicial com o seguinte comando.
$ cd ~
$ git clone https://github.com/cassou/VL53L0X_rasp
Em seguida, vá para o diretório baixado e especifique a API
( VL53L0X_1.0.2 ) salva no Raspberry Pi anteriormente para compilar.
$ cd VL53L0X_rasp
$ API_DIR = ~ / VL53L0X_1.0.2 make
Como é um grande negócio, vamos compilar e executar o programa de amostra da 
linguagem C salvo em VL53L0X_rasp.
$ API_DIR = path / to / the / api / dir faça exemplos
$ ./bin/vl53l0x_ContinuousRanging_Example


Usando VL53L0X com Python
Instale os módulos necessários para VL53L0X_rasp_python
$ sudo apt-get install build-essential python-dev

Instale VL53L0X_rasp_python
$ cd ~
$ git clone https://github.com/johnbryanmoore/VL53L0X_rasp_python.git
$ cd VL53L0X_rasp_python
$ make

Vamos executar o programa de amostra aqui também
          $ cd python
$ python VL53L0X_example.py
